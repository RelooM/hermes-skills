# Hermes Skills Repository

This repository serves as a collection of reusable Hermes skills for automating tasks and workflows. Each skill is designed to be self-contained and configurable for various use cases.

## Table of Contents
- [Overview](#overview)
- [Getting Started](#getting-started)
- [Skills List](#skills-list)
- [Contributing](#contributing)
- [License](#license)

## Overview

The repository follows a modular structure to simplify management and contribution. New skills can be added by creating a new directory under the `skills/` folder, with a clear naming convention and a `SKILL.md` file containing the skill's description and usage instructions.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/RelooM/hermes-skills.git
   cd hermes-skills
   ```
2. **Install dependencies** (if any specific to a skill)
3. **Configure skills** as per their individual documentation

## Skills List

- **bitwarden-secrets**: A skill for managing Bitwarden secrets integration with Hermes Agent.

## Contributing

Contributions are welcome! To add a new skill:
1. Create a new directory under `skills/` with a descriptive name
2. Add a `SKILL.md` file with the skill's documentation
3. Include any necessary scripts or configuration files
4. Submit a pull request for review
